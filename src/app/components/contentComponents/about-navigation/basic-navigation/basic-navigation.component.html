Если у нас имеется набор дочерних маршрутов, то возможно мы хотим открывать один из них сразу же по переходу на этот родительский. Для этого мы в модуле маршрутизации в массиве roots определяем объект с двумя свойствами: path и redirectTo.

В path мы передаем родительский маршурт, а redirectTo - маршрут куда мы хотим попасть изначально.

ВАЖНО, чтобы указанный объект был определен выше его декларирования с дочерними маршрутами.
